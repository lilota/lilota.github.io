Supported Microcontrollers

ESP32

Full support for ESP32 series with WiFi and Bluetooth capabilities

The ESP32 is a powerful, feature-rich microcontroller with integrated WiFi and Bluetooth. Ideal for IoT, automation, and sensor projects. Supports dual-core processing, low-power modes, and a wide range of peripherals.

ESP8266

Compatible with ESP8266 for cost-effective WiFi-enabled projects

The ESP8266 is a low-cost WiFi microchip with full TCP/IP stack and microcontroller capability. Perfect for simple IoT devices, smart sensors, and home automation on a budget.

Raspberry Pi Pico

Native support for RP2040-based boards with dual-core processing

The Raspberry Pi Pico (RP2040) features a dual-core ARM Cortex-M0+ processor, flexible I/O, and robust performance for advanced embedded applications. Great for education, prototyping, and real-time control.

Quick Start Guide

Prerequisites

  • Supported Microcontroller: ESP32, ESP8266, Arduino Pico
  • Computer with Linux, MacOS, or Windows
  • USB data cable (Some are power delivery only)
  • Serial programmer (for boards without built-in programmers)

Setup Methods

Terminal Method

Follow the contributing guide for terminal-based setup.

IDE Method

  1. Install the lilota IDE
  2. Use IDE to flash built files onto esp/pico
  3. After flashing, a GUI menu will pop up to configure WiFi, MQTT, etc which can be brought up later in the device settings
  4. Create and edit files which can then be flashed onto the board

Basic TCL Syntax

Comments

# This is a comment
puts 5; # This is also a comment

Variables

set led [gpio -mode out 12]
set num 5
puts $num;  # Prints 5

GPIO Control

# Create LED on pin 12
set led [gpio -mode out 12]
$led on;   # Turn LED on
$led off;  # Turn LED off

# Create button on pin 5
set button [gpio -mode in 5]
$button change {
    if { $value } {
        $led on
    } else {
        $led off
    }
}

PWM Control

# Create PWM on pin 27
set buzzer [pwm -channel 0 -resolution 8 -frequency 440 27]
$buzzer duty 0.5;  # Set 50% duty cycle

Loops and Functions

# For loop
for {set i 0} {$i < 5} {incr i} {
    puts $i
}

# Function definition
proc blinkLED {times} {
    global led
    for {set i 0} {$i < $times} {incr i} {
        $led on
        after 500
        $led off
        after 500
    }
}
